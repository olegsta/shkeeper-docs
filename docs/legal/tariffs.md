# SHKeeper Tariffs

SHKeeper offers multiple tariff plans to fit different business needs, from free open-source usage to fully customized enterprise solutions.

---

## 🔹 Open-Source Plan (Free)

- Choose which coins you need
- Select a server with suitable parameters
- Configure server following instructions
- Set up passwords and configure SHKeeper
- Create a module using the API to integrate SHKeeper into your project
- Free modules available for **WooCommerce / WordPress** and **WHMCS**
- Ideal for businesses with technical resources to self-manage setup

---

## 🔹 Advanced Plan ($2000 / per year)

- Professional integration and setup by SHKeeper team
- Instant coin implementation
- Extended technical support
- Faster updates than open-source version
- Personal customer manager
- Includes one year of support, documentation (on request), server reinstallations, and bug fixing if necessary

---

## 🔹 Custom Plan (On Request)

- Detailed product analysis
- SHKeeper customization according to your requirements
- Full integration and setup
- Extended technical support
- Personal customer manager
- Pricing is discussed individually based on your project needs
