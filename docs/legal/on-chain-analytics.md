# SHKeeper On Chain Analytics

**On Chain Analytics** refers to monitoring and analyzing blockchain transactions directly on-chain. SHKeeper allows you to **track payments and monitor wallet activity** for supported cryptocurrencies, but it **does not include advanced on-chain analytics like AML monitoring or illicit fund detection**.  

---

## What SHKeeper Provides

- **Transaction Tracking** — monitor incoming and outgoing payments in real-time.  
- **Invoice Status Updates** — automatically track payment confirmations.  
- **Multi-Coin Support** — Bitcoin, Ethereum, Litecoin, Tron, USDT, USDC, and more.  
- **Webhooks** — trigger external tools or scripts when payments occur.  

---

## Limitations

- SHKeeper does **not provide built-in compliance analytics**, risk scoring, or detection of illicit funds.  
- Advanced on-chain analytics (e.g., transaction graph analysis, sanction screening) must be implemented externally.  
- You can use SHKeeper webhooks to integrate with third-party blockchain monitoring tools.  

---

## Use Cases

- Merchants can track all payments in real-time.  
- Developers can build dashboards or alerts on top of transaction events.  
- Businesses can feed transaction data into compliance systems via webhooks.  

---

**Summary:**  
SHKeeper provides the infrastructure to capture and monitor on-chain transactions, but **advanced analytics and compliance features must be handled externally**. Integration via webhooks allows building custom monitoring and reporting solutions.
