## SHKeeper Demo

SHKeeper provides an official **demo version** so you can try the system without installing it yourself. The demo runs in a test environment and lets you explore the core options and features of the payment processor. :contentReference{index=0}

- **Demo URL:** https://demo.shkeeper.io  
- **Login:** `admin`  
- **Password:** `admin` :contentReference{index=1}

You can also watch the official video guide on YouTube that walks through the SHKeeper demo and main features:  
👉 https://www.youtube.com/watch?v=yYK_JAm1_hg :contentReference{index=2}

The demo gives you practical experience using SHKeeper’s interface and configuration options before deploying your own instance. :contentReference{index=3}
