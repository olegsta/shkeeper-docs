# SHKeeper Changelog

This changelog covers all official updates from SHKeeper.

## November 20, 2025 — Bitcoin Support via Public Node & Reduced Server Requirements

SHKeeper adds support for Bitcoin through public nodes and reduces server requirements for Bitcoin and other cryptocurrencies. [shkeeper.io](https://shkeeper.io/news?utm_source=chatgpt.com)

## November 7, 2025 — TRON Staking for Lower TRC‑20 Fees

Introduced TRON Staking, including Energy delegation, Bandwidth management, and optional Super Representative voting to reduce TRC‑20 fees. [shkeeper.io](https://shkeeper.io/news/shkeeper-introduces-tron-staking-for-lower-trc-20-fees?utm_source=chatgpt.com)

## September 8, 2025 — Ethereum Internal Transaction Detection

Detects and processes Ethereum internal transactions (smart contract-based transfers) correctly for invoice status updates. [shkeeper.io](https://shkeeper.io/news/shkeeper-update-eth-internal-transaction-detection?utm_source=chatgpt.com)
