# First Payment

This guide explains how to receive your **first cryptocurrency payment** in SHKeeper using the standard wallet payment flow. This is separate from the Invoice system — each payment uses the wallet address assigned by SHKeeper.

---

## 🔹 Steps to Receive Your First Payment

1. **Log in to SHKeeper**  
   - Access your SHKeeper dashboard.  
   - Navigate to the **Payments** section.

2. **Create a Payment**  
   - Select the **coin** (BTC, ETH, USDT, etc.).  
   - Click ** Payout** in the dashboard.  
   - Optionally, specify an **amount** or leave it flexible for the sender.  
   - Add a **Destination:**.

4. **Receive Funds**  
   - Senders transfer the cryptocurrency to the provided address.  
   - SHKeeper monitors the blockchain automatically and updates the payment status.

5. **Confirm Payment**  
   - Once the transaction is confirmed on the blockchain, SHKeeper marks the payment as **Success**.  
   - You can view the **transaction details**, **amount**, and **Crypto** in the dashboard.
