# How To Try Demo

SHKeeper provides a **demo environment** so you can quickly explore the system without performing a full installation. The demo allows you to test key features like wallet management, payment processing, and exchange integration.

---

## 🔹 Steps to Try the Demo

1. **Open the Demo URL**  
   - Navigate to the SHKeeper demo page: [https://demo.shkeeper.io/](https://demo.shkeeper.io/)

2. **Login Credentials**  
   - **Username:** `admin`  
   - **Password:** `admin`

3. **Create a Test Wallet**  
   - The demo allows you to create wallets for supported cryptocurrencies.  
   - Wallets are pre-populated with test balances for experimentation.

4. **Perform Test Transactions**  
   - Try sending and receiving payments using the demo wallets.  
   - Observe how SHKeeper handles confirmations and updates balances in real-time.

5. **Explore Features**  
   - Check the transaction history, payment links, and optional exchange integration.  
   - Experiment with different coins and simulated conversions.

---

## 📌 Notes
- Certain features may be limited or simulated for safety and testing purposes.  
- The demo is ideal for **evaluating SHKeeper capabilities** before deploying your own instance.
